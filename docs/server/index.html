Obacht Game Server

Short Description

The Obacht Game Server runs with Node.js and uses Socket.io for Realtime Communication (Websockets, if available) It handles Room Management and Communication between the Players

Installation

  • Install Node.js
  • Open Terminal, go to this directory (/server)
  • enter npm install
  • Run the server with node obacht-server.js

Requirements

The Server uses following 3rd Party Node Modules: Socket.io Backbone.js * Underscore.js
